# PowerShell Network Scanner(Сетевой сканер)

Сканер сети для обнаружения активных хостов с использованием PowerShell.

## Возможности

- **Быстрое сканирование** - многопоточная проверка 254 хостов за секунды
- **Простота использования** - одна команда для запуска
- **Легковесность** - не требует установки дополнительного ПО
- **Гибкость** - легко адаптировать под свою сеть
